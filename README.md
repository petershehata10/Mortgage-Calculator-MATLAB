
# Mortgage Calculator (MATLAB)

## Overview
This project is a MATLAB-based mortgage calculator with a graphical user interface (GUI). 
It performs automated financial calculations based on user input such as loan amount, interest rate, and duration.

## Features
- Monthly payment calculation
- Total interest calculation
- GUI-based input
- Automated computation logic
- Input validation

## Technologies Used
- MATLAB
- GUI Development
- Financial formulas implementation
